1. Ievads

Šis projekts nepieciešams tam, lai lietotāji varētu atrast konkrētu elementu tekstā. Zemāk redzams galvenais JavaScript kods, kas ieviests šajā projektā

2. JavaScript kods

Projektā tiek izmantotas divas funkcijas:
  1. Pārvaldnieka funkcija 'mainFunc()' datu iegūšanai no lietotāja, visu pārējo funkciju izsaukšanai un gala rezultātu izdrukāšanai (šī funkcija tiek aktivizēta, noklikšķinot uz pogas 'Find');
  2. funkcija vērtību testēšanai un rezultatu atrisinasanai 'checkFunc(text, letter)', lietotaja ievadits teksts un elements, kuru ir jaatrast;
Šī projekta galvenās funkcijas JavaScript kods ir parādīts laukā zemāk: 
 
const textArray = text.toLowerCase().split(''); // iterate user text
let foundLetterIndexes = new Array(); // letter indexes counter

// iterate each text letter and check
for (let x = 0; x < textArray.length; x++) {
  if (textArray[x] == letter) {
    foundLetterIndexes.push(x); // if letter is found, than add to array of found letter indexes
  }
}

// check count of found letter indexes
if (foundLetterIndexes.length >= 1) {
  if (foundLetterIndexes.length > 1) {
    result = [true, `>> Valid data: Letter is found at ${foundLetterIndexes} indexes.`]; // letter indexes count is bigger than 1
  } else {
    result = [true, `>> Valid data: Letter is found at ${foundLetterIndexes} index.`]; // letter index count is 1
  }
} else {
  result = [false, `>> Invalid data: Letter is not found.`]; // error: letter indexes count is smaller, than 1 is not found
}
